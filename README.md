stadslab-kennisbank
===================


###Database:
  - Type: MySQL
	- Name: kennisbank, kennisbank_test
	[default]
		- Username: root
		- Password: 123456
		- Port: 3307
	Check configuration on DataSource.groovy to match your system's.

###Checkouts:

    Checkouts are saved in /var/stadslab/checkouts.
    Make sure the directory exists and has the right permissions.
